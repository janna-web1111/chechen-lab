# Спецификация чтения и историй

Источник: адаптировано из найденных спецификаций `chechen-kids-app-specs/specs/feature-spec-reading.md`.

## 1. Назначение документа

Этот документ описывает модуль чтения: список историй, читалку, подсказки, перевод и словарь.

## 2. Описание модуля

Модуль чтения дает пользователю короткие истории на чеченском языке. При наведении или нажатии на слово приложение показывает перевод-подсказку. У истории также должен быть полный перевод и список новых слов.

## 3. Пользовательский поток

```text
Главный экран -> Чтение -> Список историй -> Читалка
```

Пользователь должен:

- открыть список историй;
- выбрать историю;
- читать чеченский текст;
- получать подсказку к отдельным словам;
- открыть полный перевод;
- посмотреть словарь истории;
- вернуться к списку историй.

## 5. Формат данных

```text
Story:
  id: number
  level: string
  title: string
  body: StoryToken[]
  translation: TranslationMap
  vocab: VocabularyItem[]

StoryToken:
  word?: string
  tip?: string
  text?: string
```

Пример токенов:

```text
{ word: "Нана", tip: "Мать" }
{ text: ". " }
```

## 6. Требования к читалке

- Чеченский текст должен быть главным визуальным элементом.
- Подсказка должна открываться без перехода на другой экран.
- В веб-версии подсказка должна работать по клику и через focus.
- Hover можно использовать как дополнительное поведение для desktop, но не как единственный способ открыть подсказку.
- Полный перевод должен быть доступен отдельно от подсказок.
- Словарь истории должен показывать новые или важные слова.

## 7. Улучшения

- больше историй: минимум 5 на уровень;
- вопросы на понимание после прочтения;
- режим чтения по предложениям;
- отметка прочитанных историй;
- добавление историй учителем или редактором.
