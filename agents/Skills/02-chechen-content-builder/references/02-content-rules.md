# Правила учебного контента

## Форма контента MVP

Целевой путь: A0 -> A1.

Для MVP нужны:

- 5 первых тем.
- 8-12 карточек на тему, если пользователь не указал другое число.
- 5-10 вопросов квиза на тему.
- Русские переводы для MVP.
- Опциональная транскрипция или подсказка чтения.
- Статус проверки чеченского контента.

## Рекомендуемые поля данных

Тема:

- `id`
- `level`
- `order`
- `title`
- `description`
- `lessons`
- `cards`
- `quiz`
- `status`

Карточка:

- `id`
- `themeId`
- `order`
- `ce`
- `ru`
- `transcription`
- `readingHint`
- `difficulty`
- `verificationStatus`

Вопрос квиза:

- `id`
- `cardId`
- `prompt`
- `options`
- `correctOptionId`
- `explanation`

## Статусы проверки

Использовать один из статусов:

- `draft`
- `needs_native_review`
- `reviewed`
- `published`

Использовать `reviewed` или `published` только если пользователь подтвердил проверку.

## Чеклист проверки носителем

- Чеченское написание корректно.
- Русский перевод подходит для beginner-уровня.
- Транскрипция или подсказка чтения соответствует произношению.
- Слово или фраза подходят для A0 -> A1.
- Грамматическое объяснение не вводит в заблуждение.
- Культурный контекст уважительный и точный.

## Границы контента

- Избегать сложной грамматики в MVP.
- Избегать длинных уроков.
- Избегать неподтвержденных заявлений о произношении.
- Избегать редких слов, если они не помогают цели первых 10 минут.
- Делать примеры полезными для детей, семей диаспоры, взрослых новичков и иностранцев.
