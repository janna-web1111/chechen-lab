# Контент MVP: 5 первых тем

## 1. Назначение документа

Этот документ фиксирует первые 5 тем MVP и структуру учебного контента для подготовки карточек и квизов.

Чеченские слова, транскрипции и подсказки по чтению должны быть добавлены или проверены носителем языка либо ответственным проверяющим. До проверки статус контента: `needs_native_review`.

Внешние источники для части чернового контента:

- русско-чеченский разговорник Chechnya Info (`https://chechnyainfo.com/russko-chechenskiy-razgovornik.html`);
- страница переведенных слов и фраз Govzalla (`https://govzalla.com/dosh/translated?page=32`) как источник кандидатов для коротких фраз.
- чеченско-русский словарь А. Г. Мациева (`https://ps95.ru/wp-content/uploads/2018/07/Maciev_A.G_Chechensko-russkiy_slovar.pdf`) как дополнительный словарный источник для сверки отдельных базовых слов и символа `Ӏ`.

Материал из внешних источников не считается опубликованным без проверки носителем.

## 2. Общие правила

- Уровень: A0 -> A1.
- Язык интерфейса и переводов: русский.
- Размер темы: 8 карточек для первой версии.
- Размер квиза: 5 вопросов для первой версии.
- Порог прохождения квиза: 70%.
- Формат вопросов: выбор одного правильного ответа.
- Повторное прохождение квиза: разрешено без ограничений.
- Тема считается завершенной, если пользователь просмотрел все карточки темы и прошел квиз.

## 5. Квизы

Для каждой темы первый квиз строится по одному шаблону:

- 5 вопросов;
- каждый вопрос связан с одной карточкой;
- пользователь видит чеченский текст и выбирает русский перевод;
- 1 правильный вариант и 3 неправильных;
- варианты берутся из карточек той же темы, если возможно;
- после ответа показывается короткая обратная связь.

### Шаблон вопроса

```json
{
  "id": "greetings-q01",
  "cardId": "greetings-01",
  "prompt": "Выберите перевод",
  "options": [
    { "id": "a", "text": "доброе утро (к одному человеку)" },
    { "id": "b", "text": "добрый день (к одному человеку)" },
    { "id": "c", "text": "добрый вечер (к одному человеку)" },
    { "id": "d", "text": "спокойной ночи (к одному человеку)" }
  ],
  "correctOptionId": "a",
  "explanation": "Правильный ответ: доброе утро (к одному человеку)."
}
```

## 6. Чеклист проверки

Перед публикацией каждой темы нужно проверить:

- чеченское написание;
- русский перевод;
- подсказку по чтению;
- соответствие уровню A0 -> A1;
- отсутствие слишком сложной грамматики;
- отсутствие дублей между карточками одной темы;
- корректность вариантов ответа в квизе.

## 7. Кандидаты из внешних источников

Фразы с Govzalla page 32, которые можно рассмотреть для будущей темы короткого знакомства или бытовых просьб:

- `Как тебя зовут?`
- `Можно кое-что спросить?`
- `Дай воды`

Эти фразы пока не добавлены в карточки MVP, чтобы не смешивать тему приветствий с вопросами и просьбами.

## 8. Зафиксированные решения и ограничения

- Ответственный проверяющий чеченского контента должен быть назначен до публикации учебного материала как `published`.
- В первой версии используется 8 карточек на тему.
- Дубль `вода` в темах `home` и `food` допустим как временный черновик, но должен быть проверен или заменен при контентной проверке.
- Отдельные короткие фразы после первых слов относятся к post-MVP, если они увеличивают объем первой проверки.
- До проверки носителем языка чеченский текст остается в статусе `needs_native_review`, даже если он уже заполнен из внешнего источника.
- Черновой контент можно менять по внешним источникам для подготовки review, но это не переводит карточку в `reviewed` или `published`.
