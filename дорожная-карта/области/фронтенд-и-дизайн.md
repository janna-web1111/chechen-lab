# Roadmap: frontend и дизайн

Здесь фиксируются завершенные изменения во frontend:

- экраны;
- маршруты;
- компоненты;
- пользовательские потоки;
- состояние приложения;
- ошибки и пустые состояния.

Здесь же фиксируются завершенные изменения в интерфейсе:

- визуальные правила;
- layout;
- адаптивность;
- UI-состояния;
- UX-решения;
- изменения по удобству первого пути.

Новые записи добавляются сверху.
