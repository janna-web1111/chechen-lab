# Спецификация игровых модулей

Источник: адаптировано из найденных спецификаций `chechen-kids-app-specs/specs/feature-spec-*.md`.

## 1. Назначение документа

Этот документ детализирует учебные и игровые модули, которых не хватало в общей функциональной спецификации:

- карточки;
- квиз;
- сопоставление;
- правописание;
- охота за словами.

## 2. Карточки

### Описание

Карточки - основной способ изучения новых слов. Пользователь видит слово на языке интерфейса, нажимает на карточку и получает чеченский перевод с транскрипцией.

### Поток

```text
Главный экран -> Карточки -> Выбор уровня -> Просмотр карточек -> Квиз
```

### UI-компоненты

| Элемент | Назначение |
|---------|------------|
| Карточка | Контейнер с передней и обратной стороной |
| Передняя сторона | Слово на языке интерфейса |
| Обратная сторона | Чеченское слово и транскрипция |
| Назад | Возврат к выбору уровня |
| Предыдущее | Переход к предыдущему слову |
| Следующее | Переход к следующему слову |
| Счетчик | Позиция карточки: `X / Y` |
| Квиз | Запуск квиза по текущему уровню |

### Логика

```text
startFlashcards:
  1. Сбросить индекс слова.
  2. Сбросить состояние переворота карточки.
  3. Установить текущий уровень.
  4. Показать первое слово.

updateFlashcard:
  1. Получить слово из vocabulary[currentLevel][currentWordIndex].
  2. Показать перевод на языке интерфейса.
  3. Показать чеченское слово и транскрипцию на обратной стороне.
  4. Обновить счетчик.
```

### Улучшения

- изображения для слов уровня A0;
- случайный порядок карточек;
- отметка изученных слов;
- spaced repetition.

## 3. Квиз

### Описание

Квиз проверяет знание слов. Пользователь видит слово на языке интерфейса и выбирает правильный чеченский перевод из вариантов.

### Поток

```text
Главный экран -> Квиз -> Выбор уровня -> 10 вопросов -> Результат
Карточки -> Квиз -> 10 вопросов -> Результат
```

### UI-компоненты

| Элемент | Назначение |
|---------|------------|
| Вопрос | Текст вопроса |
| Варианты | 4 кнопки с чеченскими словами |
| Обратная связь | Сообщение о правильности ответа |
| Счет | Количество правильных ответов |
| Далее | Переход к следующему вопросу |

### Логика

```text
startQuiz:
  1. Перемешать слова текущего уровня.
  2. Выбрать 10 слов в quizPool.
  3. Сбросить индекс вопроса и счет.
  4. Показать первый вопрос.

showQuizQuestion:
  1. Взять текущее слово.
  2. Сформировать 1 правильный и 3 неправильных варианта.
  3. Перемешать варианты.
  4. Показать вопрос.

handleQuizAnswer:
  1. Сравнить выбранный ответ с правильным.
  2. Обновить счет и обратную связь.
  3. Разблокировать переход дальше.
```

### Генерация вариантов

- правильный ответ берется из текущего слова;
- неправильные ответы берутся из того же уровня;
- правильный ответ не должен дублироваться среди неправильных;
- порядок вариантов должен перемешиваться.

## 4. Сопоставление

### Описание

Игра на поиск пар: пользователь сопоставляет слово на языке интерфейса с чеченским переводом.

### Поток

```text
Главный экран -> Сопоставление -> Выбор уровня -> Игра -> Результат
```

### UI-компоненты

| Элемент | Назначение |
|---------|------------|
| Доска | Сетка карточек |
| Счет пар | Например: `Пары: 0 / 5` |
| Таймер | Время прохождения |
| Результат | Итоговое сообщение |

### Логика

```text
startMatching:
  1. Выбрать 5 случайных слов текущего уровня.
  2. Создать 10 карточек: 5 переводов и 5 чеченских слов.
  3. Перемешать карточки.
  4. Запустить таймер.
  5. Сбросить selectedCard и matchedPairs.

handleCardClick:
  1. Игнорировать уже найденные пары.
  2. Если первая карточка не выбрана, выбрать текущую.
  3. Если первая карточка выбрана, сравнить пару.
  4. При совпадении пометить обе карточки как найденные.
  5. При ошибке показать обратную связь и сбросить выбор.
  6. После всех пар показать результат.
```

### Улучшения

- выбор количества пар;
- сохранение лучшего времени;
- звуковые эффекты;
- режим с обратным отсчетом.

## 5. Правописание

### Описание

Модуль тренирует написание чеченских слов. Пользователь видит перевод на языке интерфейса и вводит чеченское слово вручную.

### Поток

```text
Главный экран -> Правописание -> Выбор уровня -> 5 слов -> Результат
```

### UI-компоненты

| Элемент | Назначение |
|---------|------------|
| Слово-задание | Слово на языке интерфейса |
| Поле ввода | Ответ пользователя |
| Подсказка | Первые буквы правильного ответа |
| Проверить | Проверка ответа |
| Пропустить | Переход к следующему слову |
| Обратная связь | Правильно или неправильно |
| Прогресс | Например: `1 / 5` |
| Счет | Количество правильных ответов |

### Логика

```text
startSpelling:
  1. Выбрать 5 случайных слов текущего уровня.
  2. Сбросить индекс и счет.
  3. Показать первое слово.

checkSpelling:
  1. Нормализовать ввод: trim и lowercase.
  2. Нормализовать правильный ответ.
  3. Сравнить ответ.
  4. Обновить счет и обратную связь.
  5. После последнего слова показать результат.

showHint:
  1. Каждое нажатие открывает еще одну букву.
  2. Подсказка не должна сразу раскрывать все слово.
```

### Правила сравнения

- сравнение без учета регистра;
- пробелы в начале и конце игнорируются;
- чеченские специальные символы должны учитываться;
- в будущем можно добавить мягкую проверку опечаток.

## 6. Охота за словами

### Описание

Быстрая игра в обратном направлении: пользователь видит чеченское слово с транскрипцией и выбирает правильный перевод.

### Поток

```text
Главный экран -> Охота за словами -> Выбор уровня -> 8 раундов -> Результат
```

### UI-компоненты

| Элемент | Назначение |
|---------|------------|
| Чеченское слово | Основной стимул вопроса |
| Транскрипция | Произношение латиницей |
| Варианты | Кнопки с переводами |
| Обратная связь | Правильно или неправильно |
| Прогресс | Например: `1 / 8` |
| Счет | Количество правильных ответов |

### Логика

```text
startWordHunt:
  1. Перемешать слова текущего уровня.
  2. Выбрать 8 слов.
  3. Сбросить индекс и счет.
  4. Показать первый вопрос.

showWordHuntQuestion:
  1. Показать чеченское слово и транскрипцию.
  2. Сформировать 1 правильный и 3 неправильных перевода.
  3. Перемешать варианты.

handleWordHuntAnswer:
  1. Сравнить выбранный перевод с правильным.
  2. Обновить счет и обратную связь.
  3. Автоматически перейти к следующему вопросу.
  4. После 8 вопросов показать результат.
```

### Отличия от квиза

| Аспект | Квиз | Охота за словами |
|--------|------|------------------|
| Направление | перевод -> чеченский | чеченский -> перевод |
| Количество вопросов | 10 | 8 |
| Транскрипция | опционально | обязательно |
| Переход | по кнопке | автоматически |
