# Chechen Lab MVP app

Статический старт веб-MVP без backend и без установки зависимостей.

## Запуск

Откройте в браузере:

```text
app/index.html
```

## Что уже есть

- стартовый экран;
- путь A0 -> A1 из 5 тем;
- карточки;
- квиз на 5 вопросов;
- результат квиза;
- локальный прогресс через `localStorage`;
- экран повторения изученных карточек.

## Контент

Учебные данные лежат в `app/content.js`.

Чеченские слова и подсказки по чтению сейчас отмечены как `TBD` и `needs_native_review`, пока их не проверит носитель языка или ответственный проверяющий.
